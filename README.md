> 基于vue.js仿去哪儿网

## 主要使用的库（框架）

- **Vue**: 实现MVVM设计模式
  - **Vue-CLI**: *使用脚手架快速建立基础结构*
  - **Vuex**: *全局数据共享*
  - **VueRouter**: *页面路由*
- **Axios**: 异步数据请求
- **Better-Scroll**: 更好的滚动效果

```sh
npm install
```

## 使用

```sh
npm run start
```
